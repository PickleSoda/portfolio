## Overview

ChatFTW was an AI-driven sports prediction platform where users could view AI-generated predictions for various sports matches and engage in discussions and live chat about the events. The platform aimed to provide sports enthusiasts with cutting-edge predictions using AI and real-time data from news sources and private APIs. Despite its initial success, the project encountered challenges with social media platform bans and was eventually discontinued. 

The platform garnered over 30,000 app users via marketing across multiple platforms, including Twitter (3.5k followers), Facebook (2k followers), and other social media channels like Instagram, TikTok, and Twitch, many of which also faced bans due to the affiliate funnel model, which was flawed from the start.

## Key Features

- **AI-Powered Predictions**: The platform scraped data from several sports news websites using **Selenium** and **Beautiful Soup**, analyzed it with AI, and provided users with sports match predictions.
- **Real-Time Chat**: Integrated a chat feature where users could discuss upcoming matches and predictions in real-time, fostering community engagement.
- **Custom Dashboard**: Designed and implemented a fully custom dashboard where users could view predictions, stats, and other key information.
- **Social Media Integration**: Managed several social media platforms (Twitter, Facebook, Instagram, TikTok, Twitch) to grow the user base, although many were banned due to the affiliate funnel model.
- **Full-Stack Development**: Built the frontend using **Vue.js** and **Pinia** for state management, while the backend relied on **Flask**. **Langchain**, **Neo4j**, and **MongoDB** were used for data storage and AI-based analysis.
- **Affiliate Funnel Model**: Initially set up as an affiliate marketing funnel where revenue was generated when users lost money on predictions, which ultimately led to its failure.

## Technologies Used

- **Vue.js**: Built the frontend to ensure a smooth, reactive user experience.
- **Pinia**: State management for handling real-time chat data and prediction results.
- **Flask**: The backend API, which managed data flow and AI analysis.
- **Selenium & Beautiful Soup**: For scraping sports data from external news websites.
- **Langchain**: Used for AI-based analysis of sports data and match predictions.
- **Neo4j & MongoDB**: Database systems for storing prediction results, user data, and graph-based relationships between sports events.
- **Atomo Analytics**: Utilized for tracking user behavior and interaction on the platform.

## Challenges and Learnings

One of the major challenges in this project was maintaining a presence on social media platforms. Many accounts were banned due to the controversial nature of the affiliate funnel model. this was also our biggest contradiction we made money when users lost it, leading to eventual shutdowns. Additionally, creating accurate sports predictions required complex data scraping and machine learning models that consumed significant computational resources.

I also learned how to implement AI solutions in a real-time platform, managing data pipelines between web scraping tools, AI models, and a responsive frontend. This project sharpened my skills in full-stack development, especially integrating complex AI algorithms with a live user interface.

## Outcome

Although ChatFTW is no longer active, it saw significant initial growth, reaching over 30,000 users. The flawed business model based on affiliate losses and the social media bans it faced led to its discontinuation. However, the project was an excellent learning experience in full-stack development, AI integration, and managing a high-traffic platform. Despite its challenges, the platform demonstrated the power of AI-driven sports predictions and provided valuable insights into managing large-scale web applications.
