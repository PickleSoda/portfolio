## Overview

Kebzari.ge is a dedicated ecommerce platform for Kebzari, a local Georgian jewelry shop renowned for its traditional and handcrafted jewelry pieces. The goal was to create an online presence that not only showcases Kebzari's exquisite products but also provides a seamless shopping experience for customers. Leveraging Shopware and Vue Storefront, the website was developed to be highly responsive and user-friendly, ensuring accessibility across various devices. The project involved creating a modular codebase with Vuex for state management and SCSS for styling, all built on top of Material UI to maintain a consistent and modern design aesthetic.

Both the Vue.js frontend and the Shopware backend were fully dockerized and separated, enabling ease of deployment and scalability for future updates and expansions.

## Key Features

- **Modular Code Design**: Implemented a scalable and maintainable code structure using Vuex for efficient state management and SCSS for organized styling.
- **Custom UI Components**: Developed custom components tailored to Kebzari's unique branding and product presentation, built on top of Material UI to ensure a cohesive look and feel.
- **Multilingual Support**: Designed the website to support multiple languages, with Georgian currently turned off, allowing for future localization as the content becomes available.
- **Progressive Web App (PWA)**: Enhanced the website's performance and user experience by enabling PWA features, making it mobile-friendly and ensuring fast load times even on slower networks.
- **Ecommerce Integration**: Utilized Shopware for robust ecommerce functionalities, including product listings, shopping cart, and secure checkout processes.
- **Content Management**: Structured the site to accommodate ongoing content creation, ensuring that new products and information can be easily added as Kebzari expands its offerings.
- **Dockerized Deployment**: Both the Vue.js frontend and Shopware backend are fully dockerized, ensuring separation of concerns, easier scalability, and seamless deployment processes.

## Technologies Used

- **Shopware**: Powered the ecommerce backend, providing a reliable and scalable platform for managing products, orders, and customer data.
- **Vue Storefront**: Facilitated the development of a responsive and high-performance frontend, ensuring a smooth user experience across all devices.
- **Vuex**: Managed the application's state efficiently, allowing for seamless data flow and component interaction.
- **SCSS**: Employed for styling, enabling the creation of modular and reusable CSS with enhanced features like variables and nesting.
- **Material UI**: Provided a solid foundation for UI components, ensuring consistency and modern design standards throughout the website.
- **PWA Technologies**: Implemented to enhance the website's performance and usability on mobile devices, offering features like offline access and push notifications.
- **Docker**: Both the frontend and backend are containerized using Docker, ensuring consistent environments across development, testing, and production.

## Challenges and Learnings

One of the main challenges was integrating Shopware with Vue Storefront to create a seamless ecommerce experience. Ensuring that the custom UI components matched Kebzari's traditional aesthetic while leveraging Material UI required a delicate balance between customization and consistency. Additionally, implementing multilingual support was complex, especially with Georgian currently turned off, necessitating a flexible architecture that can easily accommodate future language additions.

Optimizing the website for mobile devices as a PWA involved addressing performance issues and ensuring that all features worked flawlessly across different screen sizes and network conditions. Dockerizing both the frontend and backend added another layer of complexity but ultimately made the development and deployment processes more efficient and scalable.

## Outcome

The Kebzari.ge website successfully launched as a robust ecommerce platform, providing Kebzari with a strong online presence that highlights their traditional Georgian jewelry. The mobile-friendly PWA ensures that customers have a smooth and engaging shopping experience, while the modular code design facilitates easy updates and scalability as the business grows. The fully dockerized architecture ensures efficient deployment and scalability, positioning Kebzari.ge for future growth and expansion into new markets.
