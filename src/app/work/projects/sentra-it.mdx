## Overview

Sentra IT is an emerging startup designed to revolutionize how businesses understand customer sentiment across their marketing campaigns. The platform leverages advanced AI to analyze and interpret sentiment data from multiple sources, providing actionable insights that help organizations optimize their marketing strategies and improve customer engagement.

As the core contributor, I designed and implemented the entire application stack from the ground up. The solution combines a modern Next.js frontend with a sophisticated Python backend, integrated with cutting-edge AI technologies including Graph RAG and semantic similarity search. While I recently stepped away from the project to focus on full-time commitments, the architecture I established provides a solid foundation for scaling the sentiment analysis platform.

## Key Features

- **AI-Powered Sentiment Analysis**: Leveraged Vercel's OpenAI API to perform real-time sentiment analysis across marketing campaigns, extracting nuanced insights about customer perception and emotional responses.
- **Graph RAG Architecture**: Implemented a Graph Retrieval-Augmented Generation (RAG) system to map relationships between posts, users, and sentiment patterns, enabling contextual understanding of sentiment trends.
- **Semantic Similarity Search**: Built semantic search capabilities to identify similar sentiments, campaign themes, and user engagement patterns across large datasets.
- **Custom Chart Artifacts & Tools**: Developed custom artifacts and visualization tools that enable users to query and display sentiment data through interactive charts and dashboards.
- **Multi-Source Data Integration**: Created a Python API that scrapes and processes data from multiple marketing channels, converts content to embeddings, and feeds insights into the AI pipeline.
- **Real-Time Insights**: Provided campaign managers with real-time sentiment metrics, trends, and anomaly detection to inform marketing decisions.

## Technologies Used

- **Next.js**: Built the frontend application with Next.js for fast, server-side rendered performance and seamless user experience.
- **Vercel OpenAI API**: Integrated OpenAI's API through Vercel for advanced natural language processing and sentiment classification.
- **Custom Artifacts & Tools**: Developed custom tools and artifacts for data visualization, enabling users to build and customize sentiment analysis dashboards.
- **Graph RAG**: Implemented a Graph-based Retrieval-Augmented Generation system to capture relationships between entities and provide contextual AI responses.
- **Semantic Similarity Search**: Built embedding-based semantic search to identify patterns and similarities in sentiment across campaigns and user segments.
- **Python API**: Developed a Python backend API for web scraping, data preprocessing, and embedding generation for the AI pipeline.
- **Vector Embeddings**: Utilized embeddings to represent text data in a semantic space, enabling similarity comparisons and efficient retrieval.

## Challenges and Learnings

One of the primary challenges was designing a system capable of processing and analyzing large volumes of marketing data while maintaining low latency for real-time insights. Balancing the complexity of Graph RAG with the need for intuitive user interfaces required careful architectural decisions.

Another significant challenge was handling the diverse data sources used in marketing campaigns—different formats, platforms, and languages—while ensuring consistent sentiment analysis quality. Creating effective embeddings and semantic search required extensive experimentation and tuning.

Additionally, architecting a system that could scale from MVP to handling enterprise-level campaigns while maintaining data quality and performance was a complex undertaking.

Through this project, I gained deep insights into building AI-powered analytics platforms, implementing Graph RAG systems, and scaling sentiment analysis at scale. The experience reinforced the importance of thoughtful architecture when building data-intensive AI applications, and the challenges of balancing innovation with practical business needs.

## Outcome

Sentra IT established a robust foundation for AI-driven marketing sentiment analysis. The architecture I designed provides the startup with a scalable platform capable of processing diverse data sources and delivering meaningful insights to marketing teams. The combination of Graph RAG, semantic search, and real-time AI analysis positions the platform to become a valuable tool for understanding customer perception.

While I stepped away from the project to focus on other commitments, the work demonstrates the feasibility of building sophisticated AI analytics platforms using modern web technologies and demonstrates expertise in full-stack architecture for AI applications.
